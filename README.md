# jquery.swap-to-gif.js
Jquery plugin to create animated gif hover effect with img replacement

# Usage

```html
<img src='/assets/images/static.jpg' class='swap-to-gif'/>
```

```javascript
$(document).ready(function() {
    $('.swap-to-gif').swapToGif();
});
```
